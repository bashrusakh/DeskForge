# rdgen setup — upstream reference

> **Not applicable to DeskForge.** DeskForge does not use rdgen as a service.
> This file is from upstream [bryangerlach/rdgen](https://github.com/bryangerlach/rdgen)
> for anyone who wants to run the original rdgen separately.

---

To run rdgen standalone:

1. Fork `bryangerlach/rdgen`
2. GitHub fine-grained token with actions permissions
3. Secrets: `GENURL`, `ZIP_PASSWORD`
4. `docker compose up -d`

Full instructions: [original setup.md](https://github.com/bryangerlach/rdgen/blob/main/setup.md).
