# rdgen setup — upstream reference

> **Не актуально для DeskForge.** DeskForge не использует rdgen как сервис.
> Инструкция ниже — из upstream [bryangerlach/rdgen](https://github.com/bryangerlach/rdgen)
> для желающих запустить оригинальный rdgen отдельно.

---

Если нужно запустить rdgen самостоятельно:

1. Форкнуть `bryangerlach/rdgen`
2. GitHub fine-grained token с правами на actions
3. Secrets: `GENURL`, `ZIP_PASSWORD`
4. `docker compose up -d`

Подробнее: [оригинальный setup.md](https://github.com/bryangerlach/rdgen/blob/main/setup.md) в upstream.
